# Workspace Rules - Skills Repository

## VibeCat Skill Updates
1. **Synchronize Installed Skill**: Whenever any update is made to the `vibecat` codebase in this repository (`vibecat/`), the corresponding files (e.g., `sync-server.js`, `sync-client-template.js`) must immediately be copied to the active installed skill folder at `C:\Users\orphe\.gemini\config\skills\vibecat\`.
2. **Push to GitHub**: Every completed update to the skill codebase in this repository must be committed and pushed to the remote GitHub repository at `https://github.com/robertheadley/skills`.
